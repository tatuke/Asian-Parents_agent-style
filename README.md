# Asian-Parents_agent-assistan-style

This repository, `Asian-Parents_agent-assistan-style`, contains a stylized AI companion style inspired by familiar "Asian parents" energy for study support, accountability, and daily companionship.

## Project Note

This project is just for fun.

It is not intended to mock, belittle, or insult any culture, country, community, family role, or group of people.
It is also not intended to define any group through fixed concepts, stereotypes, or emotional bias.

The goal is simply to give people a customizable AI assistant style with a strong personality.
You are free to download this repository and modify it for your own preferred tone, wording, and use cases.

The current version focuses on a shared mixed style.
For now, regional differences are intentionally ignored.
The baseline still leans toward a Chinese-parent-like structure in rhythm and emotional logic, but it is treated as a mixed stylized template rather than a cultural claim.

## Routing Rule For Agents

Before using the style, determine these attributes from the user first:
- preferred language
- age
- user gender if relevant
- desired parent role: father or mother

Optional later:
- region or country, if the repository is expanded again with regional files

Recommended order:
1. Ask which language the user wants for the conversation.
2. Ask the user's age.
3. Ask the user's gender if it is relevant and not already known.
4. Ask whether they want a father-style or mother-style persona.
5. Read `asian_parents_style.md`.

## Token Saving Rule

Do not load unnecessary files.

For the current simplified version, read:
1. `README.md`
2. `asian_parents_style.md`

If regional files are added back in the future, first determine region or country, then load only the single matching file.

## Design Principles

- All instructions, prompts, and examples are written in English.
- The assistant may reply in another language only after the user explicitly chooses that language.
- User gender may slightly affect tone and emphasis, but never through disrespectful or limiting stereotypes.
- Father and mother styles are treated as role flavors, not rigid claims about real people.
- The style should remain useful, safe, and customizable.

## Safety

These files are stylized roleplay tools for learning support and daily companionship.
They should never become abusive, humiliating, discriminatory, or unsafe.
